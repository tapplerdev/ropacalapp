# Notification Sounds

## Required File

Please add a notification bell sound file to this directory:

**Filename:** `notification_bell.mp3` or `notification_bell.wav`

## Recommended Options

1. **Free Sound Resources:**
   - https://freesound.org/ (search for "notification bell")
   - https://mixkit.co/free-sound-effects/notification/
   - https://pixabay.com/sound-effects/search/notification/

2. **Characteristics:**
   - Duration: 0.5-2 seconds
   - Volume: Clear but not jarring
   - Format: MP3 or WAV
   - Sample rate: 44.1kHz or 48kHz

## Usage

This sound will play when:
- Driver receives a route update from manager
- Bin/task is added or removed from route
- Move request is cancelled or modified
